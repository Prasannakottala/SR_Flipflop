# SR_Flipflop
An SR flip-flop is a basic sequential logic circuit with two inputs: S (Set) and R (Reset), and two outputs: Q and Q̅. When S=1, R=0, the output Q=1 . When S=0, R=1, the output Q=0 . If S=R=0, the previous state is retained. The condition S=R=1 is invalid, leading to an undefined state. It’s widely used in memory storage &amp; control circuits.
